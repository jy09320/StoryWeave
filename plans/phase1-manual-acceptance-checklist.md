# Phase 1 手工验收清单

> 对齐 [`PLAN.md`](../PLAN.md) 的 Phase 1 / Phase 1.5，以及 [`plans/mvp-phase1-next-step.md`](./mvp-phase1-next-step.md) 中“补齐手工验收清单与验收记录”的执行要求。

## 验收目标
确认 StoryWeave 当前是否已经完成 MVP 第一阶段的主链路闭环，并为后续 Phase 1 收口、Wangeditor 评估、AI 深化体验与 Phase 2 规划提供可复用的人工验收依据。

## 验收前准备

### 环境准备
- 使用 [`docker-compose.yml`](../docker-compose.yml) 启动完整环境
- 确认端口：
  - 前端：`http://localhost:3001`
  - 后端：`http://localhost:8001`
  - 数据库：`localhost:5433`
- 准备至少 1 个可用 AI 配置：OpenAI 或 Anthropic

### 启动检查
1. 执行 [`docker compose up --build`](../docker-compose.yml)
2. 打开 [`http://localhost:3001`](../DOCKER.md)
3. 访问 [`http://localhost:8001/api/health`](../DOCKER.md)
4. 确认前端页面可加载、后端健康检查返回正常
