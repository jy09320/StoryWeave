# req-022 验收清单

## 使用说明

本清单用于在 `req-022` 开发过程中逐项确认交付结果，避免“看起来改了很多，但主链路不成立”。

---

## A. 全局工作台骨架

- [ ] 所有核心页面已迁移到统一 App Shell
- [ ] 左侧一级导航宽度稳定，激活态正确
- [ ] 左侧二级项目大纲可折叠/展开
- [ ] 右侧辅助抽屉默认隐藏，可正常唤起
- [ ] 上下文顶栏能按页面显示不同内容
- [ ] 折叠/展开侧栏时主区不出现明显闪跳

---

## B. Dashboard

- [ ] 首页首屏优先展示最近进展而不是宣传卡片
- [ ] 最近编辑项目可一键继续写作
- [ ] 其他项目以高密度列表或表格展示
- [ ] 创建项目入口不再占据首页中央主视觉
- [ ] 空状态与已有项目状态都表现正常

---

## C. 项目工作台

- [ ] 项目页首屏能清楚展示字数、章节数、更新状态
- [ ] 项目简介等静态信息已降级处理
- [ ] 章节管理器支持快速扫描和进入编辑器
- [ ] 章节排序逻辑可用
- [ ] 项目角色绑定路径清晰
- [ ] 世界观入口清晰可达
- [ ] 创作动态图谱已接入真实数据或已明确首版真实来源

---

## D. Tiptap 编辑器底座

- [ ] 现有章节内容能在新编辑器中正常加载
- [ ] 基础输入、删除、粘贴、撤销重做可用
- [ ] 自动保存可用
- [ ] 手动保存可用
- [ ] `Ctrl+S` 可触发保存
- [ ] 标题与正文为一体化写作体验
- [ ] 从标题进入正文时无明显视觉断裂
- [ ] 正文主区具备稳定舒适的阅读宽度

---

## E. 编辑器内智能交互

- [x] 选中文本后能出现 Bubble Menu
- [x] Bubble Menu 至少包含润色、扩写、改写、设定检查
- [x] 输入 `/` 后可唤起命令菜单
- [x] Slash Command 支持键盘选择
- [x] 输入 `@角色名` 可插入 Mention
- [x] Mention 节点视觉存在但不喧宾夺主
- [x] 悬停 Mention 时能看到角色摘要卡片

---

## F. AI 改写与 Diff 闭环

- [x] 选区 AI 操作可正常触发
- [x] AI 返回结果以内联 Diff 或等价对比方式展示
- [x] 生成中存在清晰 loading 或流式占位
- [x] `Tab` 可接受改写结果
- [x] `Esc` 可放弃改写结果
- [x] 重新生成可用
- [x] 接受后结果能稳定写回正文
- [x] 放弃后正文不被污染

---

## G. 右侧参考抽屉联动

- [x] 抽屉内可查看当前项目角色速查
- [x] 抽屉内可查看世界观词条
- [x] 抽屉内可快速发起 AI 任务
- [x] 当前章节上下文能影响抽屉展示内容
- [x] 关键词命中世界观词条时，抽屉可自动定位相关内容
- [ ] 自动联动不会造成明显干扰

---

## H. 角色库与世界观页

- [ ] 角色库已采用主从视图
- [ ] 角色库支持搜索与筛选
- [ ] 角色详情编辑区结构清晰
- [ ] 世界观页已采用结构化维护方式
- [ ] 世界观页与编辑器联动的数据结构一致
- [ ] 资产页整体观感更像创作设定文档，而不是后台表单

---

## I. AI 工具箱

- [x] AI 工具箱采用任务式布局，而不是聊天式布局
- [x] 输入区与结果区分区明确
- [x] 任务类型有清晰切换方式
- [x] 高级参数默认折叠
- [x] 结果支持对照查看
- [x] 生成历史可回看和复用

---

## J. 视觉与交互统一

- [x] 全站深色层级统一
- [x] 过重卡片、边框、阴影已显著减少
- [x] 正文区、侧栏、浮层、抽屉的层级关系清晰
- [x] 快捷键 `Ctrl+B` 可切换侧栏
- [x] 快捷键 `Ctrl+J` 可唤起 AI 指令
- [x] 保存反馈轻量且不打断
- [ ] 桌面与窄屏下没有明显重叠、溢出和错位
- [x] Zen Mode 可用

---

## K. 主链路回归

- [x] 用户可从首页进入最近项目
- [x] 用户可从项目工作台进入章节编辑器
- [x] 用户可在编辑器中写作、查看设定、调用 AI
- [x] 用户可接受或放弃 AI 改写
- [x] 用户可返回工作台继续管理章节和设定
- [ ] 上述流程中不存在阻断性 UI 跳转或状态丢失
